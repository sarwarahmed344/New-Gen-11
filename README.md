# NewGen11

Blue Lock U-20 character showcase by Mirza Ahmed Sarwar 
Hussain (AS) — 6 revealed, 5 classified. The squad is 
not yet complete.

🔗 Live site: https://newgen11-sarwar.lovable.app

## About
NewGen11 is a fan art project built around the Blue Lock 
universe. Each character is hand-drawn on A4 cartridge 
paper and presented as part of a unified squad showcase.
Built with Lovable.dev.
